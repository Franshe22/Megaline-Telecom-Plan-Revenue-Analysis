# Megaline-Telecom-Plan-Revenue-Analysis
## 📝 Project Description

This project analyzes customer behavior and revenue generation for Megaline, a telecommunications operator that offers two prepaid mobile plans: Surf and Ultimate.

The analysis is based on a dataset containing information from 500 customers during 2018, including their location, plan type, number of calls made, text messages sent, and mobile data usage.

The goal of the project is to evaluate which prepaid plan generates more revenue on average, helping the marketing department decide where to allocate the advertising budget.

## 🎯 Project Objective

The main objective of this project is to analyze customer usage patterns and determine which mobile plan is more profitable for the company.

To achieve this, the analysis includes:

Studying customer usage behavior (calls, SMS, and data consumption)

Calculating the monthly revenue generated per user

Comparing the profitability of the Surf and Ultimate plans

Applying statistical hypothesis testing to determine if the revenue differences between the plans are statistically significant

This type of analysis helps telecom companies optimize pricing strategies and marketing investments.

## 📊 Data Analysis Process

The project follows a typical data analytics workflow:

Data Cleaning

Handling missing values and formatting data

Converting usage data into monthly aggregates

Data Preparation

Calculating total monthly usage per customer

Computing extra charges when usage exceeds plan limits

Exploratory Data Analysis (EDA)

Analyzing call duration, messages, and data usage

Comparing usage patterns between plans

Revenue Calculation

Estimating monthly revenue per user based on plan rules and overage charges

Statistical Analysis

Performing hypothesis testing to evaluate revenue differences between plans
